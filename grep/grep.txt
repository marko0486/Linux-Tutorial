See file in real-time and only look for “error” word
tail -f file.txt | grep -i error

Every line that begins with a capital letter between A thru E
grep "^[A-E]" file.txt

Every line that ends with a capital letter between A thru E
grep "^[A-E]" file.txt

Find only words starting with some letters and ends with some letters
grep "^[FG]oo" file.txt

Between A-C and ends with T
grep -n "[A-C]T" file.txt

Starts with A between b and e and ends with t
grep "A[b-e]t" ls2.txt

Search for different words
grep -E -i -w "Error|Warning|logs" file.txt

Get all the processes containing XXXX
ps aux | grep httpd

Filter by Dates and hours in a log file
awk -v start_day=2018/04/11 \
-v stop_day=2018/04/11 \
-v start_time=07:40 \
-v stop_time=07:42 \
'start_day <= $8 && $9 >= start_time && $8 <= stop_day && stop_time >= $9 ' console.log
